# README

This is a really good doctors & patients database.

+ Ruby version => 3.2.2
+ System dependencies => See Gemfile

+ Config
```bash
bundle install
```
+ Just to make sure
```bash
rails db:migrate VERSION=0
```
+ Database creation
```bash
rails db:migrate
rails db:migrate:status  
```
+ Database initialization
```bash
rails db:seed
```
